# YouTube Video Downloader

A simple Python tool to download YouTube videos using yt-dlp.

## Features

- Download videos from YouTube
- Supports up to 720p resolution
- Automatically merges audio and video using ffmpeg

## Requirements

- Python 3
- yt-dlp
- ffmpeg

## Installation

Install dependencies:

pip install -r requirements.txt

## Usage

Run the script:

python downloader.py

Then paste the YouTube video link.

## Example

Enter YouTube video URL:
https://youtu.be/example
